

# Stellar Cyber Dashboard & Report Library

[](https://stellarcyber.ai/)
Author: Davidson Brito - Sr Sales Engineer Brazil

This repository serves as a centralized collection of custom dashboards and reporting templates for the **Stellar Cyber Open XDR** platform. These resources are designed to help security analysts, SOC managers, and engineers visualize telemetry, detect threats faster, and automate compliance reporting.

## 📋 Overview

The Stellar Cyber platform allows for deep customization of data visualization. This project provides pre-configured JSON templates and configurations that can be imported directly into your environment to monitor specific use cases like Ransomware, Cloud Security, and User Behavior.

-----

## 🛠️ How to Import

Follow these steps to add these dashboards to your Stellar Cyber instance:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/*yourusername*/stellarcyber-dashboard-report.git
    ```
2.  **Navigate to Stellar Cyber:**
      * Log in to your Stellar Cyber DP (Data Processor).
      * Go to **Dashboard** \> **Creation** \> **+Create**.
3.  **Import Template:**
      * Click on the **Import Dashboard** button
      * Select the `.json` file from the `dashboards/` folder of this repo.
4.  **Verify Data Sources:**
      * Ensure your indices and sensors are correctly mapped to the dashboard filters.
